How to securely contact
The Global Mail

The Global Mail is serious about protecting our sources. If you have information or documents that you think the public needs to see, please read the information below to ensure that you can reach out to us without compromising your privacy.

If you're not sure which of these many communication channels fits your circumstances, refer to our secure communication flow chart at the bottom of this page.

3. The TGM Vault

We have set up the TGM Vault to make it as easy as possible for you to correspond with us securely. TGM Vault is an installation of SecureDrop — an open-source whistleblower submission system managed by Freedom of the Press Foundation for media organisations to use to securely accept documents from anonymous sources. Your messages and documents are sent over the secure Tor browser, and will only be readable by us.

To use the SecureDrop vault, you need to download the Tor Browser Bundle, then open the Tor browser and copy and paste in this special Tor URL:

http://6vhfob5xhued4dym.onion

Before making any submissions, we recommend that you carefully read the official step-by-step instructions for SecureDrop. For more information on the Vault itself, be sure to read our post on why we’ve set it up.

4. Secure email using Pretty Good Privacy (PGP)

We strongly advise against sending sensitive information through normal email. If you choose to send us an email, you should first encrypt your message using Pretty Good Privacy (PGP), with The Global Mail’s public key.

PGP works by generating pairs of public and private “keys”. The content of an email encrypted using The Global Mail’s public key, for example, can only be read using our corresponding private key, which we keep secret. It turns your email into coded gibberish that only we can decipher.

To get started using PGP on Mac OS X or Windows, we recommend this great guide put together by the Lifehacker blog.

Send the email from an account that is not tied to your identity, i.e. not your personal or work accounts; setting up a new account if necessary. To add an extra layer of security, make sure you are not using your home connection when sending the information; do so from an internet cafe or somewhere with public internet access.

We don't recommend this method for sending digital documents or email attachments. For more on sending documents, refer to the You have: Documents section below.

If your files are large – over 100 megabytes or so – uploading via TGM Vault will be slow. In order to obscure your computer’s identity, Tor sends your data through several intermediary networks, making file transfers much more cumbersome. For large files, use the TrueCrypt method outlined below.

2. TrueCrypt

If you have a large set of files, or are skeptical about transferring sensitive information through the web, we recommend using the free software TrueCrypt to encrypt files on your own computer before sending them to us.

After encrypting the documents with TrueCrypt, you will transfer these secure files to a CD, DVD or USB, and send them to us by post. These files can only be read with a password that is created when you encrypt them; you will send us this password as well – separately from the files themselves – through one of the secure communication channels detailed above. Der Spiegel has an excellent write-up on using TrueCrypt to send information safely.
